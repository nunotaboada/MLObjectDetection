This project delivers a deep learning-based solution for object detection, tailored for autonomous driving applications using the YOLOv11n model. It includes utilities for dataset augmentation, label validation, and training a lightweight YOLOv11 nano model to detect objects in images with high efficiency. Implemented by `Team07 <https://github.com/orgs/SEAME-pt/teams/team07>`_
